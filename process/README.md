# process

Records of trial and consideration that led to adopting β.

**Rejected ≠ failed** — they're part of the loop that was spun. Kept here as a reference for future design loops.

## exploration journey

| Stage | What happened | Conclusion |
| --- | --- | --- |
| 1. Concept divergence | Three concepts: spiral / initial K / node branches | Felt good about A (spiral) and C (nodes) |
| 2. Adding a brand axis | Tried embedding a personal kanji as the brand axis | Rejected — "felt too much like a name label" |
| 3. Character pivot | Realized "too serious" / "too pulled by that axis" | Pivoted toward lightness and play |
| 4. Re-presenting ranges | Five new directions: A〜E (character / music / pixel / bath / organic geometry) | Picked B (music) and C (pixel) |
| 5. Four prototypes | α (Pixel Wave) / β (Bouncing Loop) / γ (EQ Bars) / δ (Wave Ring fusion) | Adopted β |
| 6. Fusion experiment | Reworked δ three times (brightness gradient → comet → 2-peak ring) | None fully integrated β/α's strengths; settled on β alone |

## rejected variants

### [`alpha-wave.svg`](./alpha-wave.svg) — Pixel Wave

A horizontal waveform (music × data flow). Direction was strong, but completeness as a logo was weak compared to a ring's symbolic strength. Not adopted.

### [`gamma-eq.svg`](./gamma-eq.svg) — EQ Bars

A direct DAW meter. The strongest "Ableton" feel of all candidates, but the "music genre" association overshadowed personal-brand uniqueness. Not adopted.

### [`delta-wave-ring.svg`](./delta-wave-ring.svg) — Wave Ring (fusion attempt)

The final form of three attempts to fuse α × β. 12-node ring + brightness modulated as a horizontal wave + peaks on left and right poles. Trail of attempts:

1. Brightness gradient only → "looks like a uniform circle"
2. Comet (half-ring + head) → "loses both α's and β's strengths"
3. 2-peak ring (the final attempt) → barely works, but inferior to β's purity. Settled on β alone.

## learnings (for future design loops)

- **Don't anchor too hard on a brand axis** — a personal kanji axis was tried and immediately removed for feeling like a name label. Heavy meanings work better as quiet undertones.
- **Tradeoff between abstraction and identity** — pure concepts make a generic AI logo; pure identifiers make a name label. The sweet spot is in between.
- **"Uniform" and "movement" rarely coexist** — keeping ring completeness while adding motion is hard at pixel granularity. Movement worked best when concentrated in a single accent point.
- **Don't fuse blindly** — when both A and B are good, forcing a fusion can dilute both. Role separation (primary / secondary) sometimes works better — same structure as the A/B rule in `typography.md`.
